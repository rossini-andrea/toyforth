# ToyForth

ToyForth is an interpreted implementation of the Forth programming language. It does not aim for compliance nor completeness.

This project exists because antirez asked his followers to implement
it during his C language lessons.

## Currently supported words

* Numeric `int` literals
* `+-*/` operators
* Comparison operators
* `if`, `else` and `then`
* `.` dot printing operator
* `cr` prints a new line
* `dup` and `drop`

## AI usage

No LLMs were harmed in the making of this project. All code is hand crafted, except when stated.
